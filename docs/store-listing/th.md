แชทสด YouTube ซ้อนทับบนวิดีโอเต็มหน้าจอของคุณ อ่านข้อความ โพสต์ความคิดเห็น และส่ง Super Chat — โดยไม่ต้องออกจากโหมดเต็มหน้าจอ

คุณสมบัติ:
💬 แชทในโหมดเต็มหน้าจอ — โพสต์ความคิดเห็นและ Super Chat จากโอเวอร์เลย์
🖱️ ลาก ปรับขนาด จัดตำแหน่ง — วางหน้าต่างแชทไว้ตรงที่คุณต้องการ
🎨 ปรับแต่งตามสไตล์คุณ — พื้นหลัง สีตัวอักษร ขนาดฟอนต์ เบลอ ระยะห่าง
📋 บันทึกพรีเซ็ต — สลับสไตล์สำหรับสตรีมต่าง ๆ ได้ในคลิกเดียว
🌗 ธีมสว่างและมืด — ตามระบบของคุณ หรือตั้งค่าเอง
📼 รองรับอาร์ไคฟ์ — ใช้งานได้กับการเล่นซ้ำแชทในสตรีมที่บันทึกไว้
🌐 รองรับกว่า 50 ภาษา

วิธีใช้งาน:
1. ติดตั้งส่วนขยาย
2. เปิดสตรีมสด YouTube หรืออาร์ไคฟ์ที่มีการเล่นซ้ำแชท
3. เข้าสู่โหมดเต็มหน้าจอ — เปิดสวิตช์ที่มุมขวาล่าง
4. ลาก ปรับขนาด และปรับแต่งตามใจชอบ
