YouTube 라이브 채팅을 전체화면 영상 위에 오버레이로 표시합니다. 메시지를 읽고, 댓글을 쓰고, Super Chat을 보내세요 — 전체화면을 나가지 않아도 됩니다.

기능:
💬 전체화면에서 채팅 — 오버레이에서 바로 댓글과 Super Chat을 보낼 수 있습니다.
🖱️ 드래그, 크기 조절, 위치 변경 — 채팅 창을 원하는 곳에 배치하세요.
🎨 나만의 스타일 — 배경, 글자 색, 글꼴 크기, 블러, 간격을 자유롭게 설정.
📋 프리셋 저장 — 스트림에 맞는 스타일을 원클릭으로 전환.
🌗 라이트 & 다크 테마 — 시스템 설정에 따르거나 수동으로 선택.
📼 아카이브 지원 — 다시보기의 채팅 리플레이에서도 작동합니다.
🌐 50개 이상의 언어 지원.

사용 방법:
1. 확장 프로그램을 설치합니다.
2. YouTube 라이브 스트림 또는 채팅 리플레이가 있는 아카이브를 엽니다.
3. 전체화면으로 전환 — 오른쪽 하단의 스위치를 켜세요.
4. 드래그하고, 크기를 조절하고, 나만의 채팅 환경을 만드세요.
