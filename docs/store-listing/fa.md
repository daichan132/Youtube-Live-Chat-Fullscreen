🎨 ظاهر را تنظیم کنید — پس‌زمینه، رنگ متن، اندازه فونت، تاری، فاصله‌گذاری.
📋 ذخیره پیش‌تنظیم‌ها — با یک کلیک بین سبک‌های مختلف برای پخش‌های مختلف جابه‌جا شوید.
🌗 پوسته روشن و تاریک — از تنظیمات سیستم پیروی می‌کند یا دستی تنظیم کنید.
📼 پشتیبانی از آرشیو — روی پخش‌های آرشیوشده با بازپخش چت هم کار می‌کند.
